## 📌 Featured Projects

### Multi-SWE-Bench LLM Evaluation Pipeline
Long-horizon evaluation pipeline that benchmarks LLM agents against 239 real-world Rust bug-fixing tasks. Uses OpenHands agent harness for autonomous code edits, tool calls, and test execution inside Docker containers. Computes Pass@K (k=1,3,5) metrics with automated failure classification and fully reproducible benchmarking. The kind of project where you learn exactly how good (and how bad) LLMs actually are at writing code.

### TCUEOL Production Management Platform *(work project)*
Full-stack production management system (Flask + React + MongoDB) running multi-stage device testing workflows. Role-based access, EOL report generation, Arduino test jig integration, and multi-protocol hardware testing (Serial, BLE, TCP) with a PyQt5 desktop client. Deployed with Docker.

### ZenTrack Desktop Configuration Tool *(work project)*
PyQt5 desktop app with JSON-driven dynamic UI for configuring IoT devices across Serial, USB, BLE, and TCP. Custom binary protocol with CRC-8 integrity checks, command queuing with retries, and a modular transport abstraction layer. The firmware talks, and this tool makes sure it says the right things.

### [Supa Social App](https://github.com/07Kaustubh/supa-social-app)
React Native + Expo social platform with Supabase backend. Real-time posts, push notifications, media uploads, full auth flow.

### [EasifyMart](https://github.com/07Kaustubh/Major)
